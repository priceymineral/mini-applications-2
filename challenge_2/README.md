# Historical Bitcoin Price Index Chart

## Description

Web application that consumes Coindesk API to display historical Bitcoin Price Index (BPI) in a beautiful chart.

## Getting Started

### Tools/Dependencies

* React
* Express
* JavaScript
* Axios
* Webpack
* Babel
* Chart.js
* NodeJS
* Redux

### Installing
* Clone repository
* Install dependencies
```
$ git clone <REPOURL>
$ npm install
```
### Executing program

* Webpack compile
* Run the server
```
$ npm run watch
$ npm start
```
